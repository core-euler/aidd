# Известные проблемы и решения

## Таймзоны и дедлайны
**Проблема:** дедлайны сохранялись без UTC‑нормализации и воспринимались как локальные.
**Решение:** хранить дедлайны в UTC и конвертировать в локальный TZ при отображении.
**Документы:** DATA_MODEL.md, SCHEDULER.md, INFRA.md.

## Обработка callback‑кнопок
**Проблема:** некоторые callback‑кнопки не обрабатывались из‑за отсутствующих обработчиков.
**Решение:** убедиться, что для каждого callback_data есть обработчик.
**Документы:** ANTIPATTERNS.md, UI_TEXTS.md.

## Ошибки FSM при выборе сложности/дедлайна
**Проблема:** кнопки сложности/дедлайна не работали при несовпадении FSM‑состояния.
**Решение:** обрабатывать callback независимо от состояния и валидировать state внутри.
**Документы:** ANTIPATTERNS.md, FLOWS.md.
