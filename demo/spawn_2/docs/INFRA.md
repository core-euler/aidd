# Инфраструктура и окружение

## Запуск
- Приложение запускается через Docker Compose
- База данных PostgreSQL работает в отдельном контейнере

## Переменные окружения
- Токен бота: обязателен, хранится только в окружении
- Строка подключения к базе данных: обязателена
- Часовой пояс пользователя для ввода и отображения дедлайнов
- Уровень логирования

## Технологии
- Python + aiogram для работы с Telegram
- SQLAlchemy для доступа к данным

## Логирование
- Логируются все действия пользователя
- Логи включают создание задач, выполнение, удаление, просрочку и уведомления

## Надёжность
- Данные пользователя сохраняются при перезапуске
- Ошибки обрабатываются с понятным сообщением пользователю

## Безопасность
- Пользователь видит только свои данные
- Нет доступа к задачам и статистике других пользователей
