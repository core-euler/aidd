# Экраны и тексты

## Приветствие
Содержит:
- название бота
- краткое описание механики
- стартовые характеристики персонажа
- кнопку создания первой задачи

## Главное меню
Содержит:
- название бота
- краткий статус персонажа
- кнопки: новая задача, мои задачи, персонаж, статистика

## Новая задача
Шаг 1. Ввод названия:
- просьба ввести название
- проверка на пустое значение и длину

Шаг 2. Выбор сложности:
- варианты: лёгкая, средняя, сложная, эпическая

Шаг 3. Выбор дедлайна:
- быстрые варианты: через 1 час, через 3 часа, сегодня вечером, завтра утром, завтра вечером
- вариант ручного ввода

## Мои задачи
Содержит:
- список активных задач
- каждая задача показывается отдельной кнопкой
- кнопка просмотра просроченных задач
- при отсутствии задач показывается информативное сообщение

## Экран задачи
Содержит:
- название
- сложность
- дедлайн
- оставшееся время
- кнопки: выполнено, удалить, назад

## Персонаж
Содержит:
- уровень, опыт и здоровье
- визуальные прогресс-бары
- количество активных задач
- ближайший дедлайн
- кнопку возврата в меню

## Статистика
Содержит:
- количество выполненных задач
- количество просроченных задач
- процент успешности
- максимальный достигнутый уровень
- кнопку возврата в меню

## Служебные сообщения
- уведомление о просрочке и нанесённом уроне
- уведомление о достижении уровня
- уведомление о смерти персонажа
- напоминание за 1 час до дедлайна
