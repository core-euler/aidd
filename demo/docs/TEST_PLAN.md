# План тестирования (TDD)

## Создание задачи
- Проверить успешное создание задачи с валидными названием, сложностью и дедлайном
- Проверить отказ при пустом названии
- Проверить обрезку названия длиннее 200 символов и уведомление пользователя
- Проверить отказ при дедлайне в прошлом
- Проверить запрос повторного ввода при неверном формате даты

## Просмотр задач
- Проверить вывод списка активных задач
- Проверить корректную сортировку по дедлайну
- Проверить сообщение при отсутствии активных задач

## Выполнение задачи
- Проверить начисление опыта при выполнении активной задачи до дедлайна
- Проверить отсутствие опыта при выполнении просроченной задачи
- Проверить изменение статуса задачи на выполненную

## Удаление задачи
- Проверить удаление без начисления опыта и урона
- Проверить невозможность повторных действий над удалённой задачей

## Просрочка и урон
- Проверить нанесение урона при наступлении дедлайна
- Проверить единоразовость урона по задаче
- Проверить суммирование урона при одновременной просрочке нескольких задач

## Смерть персонажа
- Проверить сброс уровня, опыта, здоровья и максимального здоровья при смерти
- Проверить удаление всех активных задач при смерти
- Проверить уведомление пользователя о смерти

## Уровни и опыт
- Проверить повышение уровня при достижении порога
- Проверить перенос излишка опыта
- Проверить несколько повышений уровня за одно действие

## Напоминания
- Проверить отправку уведомления за 1 час до дедлайна
- Проверить, что напоминание отправляется только один раз
- Проверить отсутствие напоминания для выполненных и просроченных задач

## Статистика
- Проверить увеличение счетчиков выполненных и просроченных задач
- Проверить расчёт процента успешности
- Проверить хранение максимального достигнутого уровня
