# Этапы реализации

## Этап 1: Базовый бот
Результат:
- /start создаёт персонажа
- Главное меню через inline‑кнопки
- Экран персонажа

Проверка:
- Запуск бота
- Навигация по меню
- Корректный вывод статуса персонажа

## Этап 2: Управление задачами
Результат:
- Пошаговое создание задачи
- Список задач кнопками
- Детали задачи с кнопками действий

Проверка:
- Создание задач с разными дедлайнами
- Переходы между экранами
- Удаление задачи

## Этап 3: Игровая механика
Результат:
- Начисление XP и уровни
- Просрочка задач и урон
- Суммарный урон при нескольких просрочках

Проверка:
- Выполнение задач разной сложности
- Повышение уровня
- Просрочка задач

## Этап 4: Смерть и уведомления
Результат:
- Смерть персонажа при 0 HP
- Удаление активных задач
- Уведомления за час, просрочка, уровень, смерть

Проверка:
- Сценарии смерти
- Проверка отправки уведомлений

## Этап 5: Статистика и polish
Результат:
- Экран статистики
- Прогресс‑бары
- Обработка edge cases

Проверка:
- Статистика соответствует данным
- Все edge cases из SPEC.md
