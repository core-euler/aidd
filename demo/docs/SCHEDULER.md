# Планировщик и уведомления

## Проверка дедлайнов
- Периодичность: каждые 5 минут
- Обрабатываются только активные задачи
- Дедлайны сравниваются в UTC
- Если дедлайн прошёл, задача становится просроченной, наносится урон, пользователь уведомляется
- Урон наносится один раз на задачу

## Напоминания
- Уведомление отправляется за 1 час до дедлайна
- Уведомление отправляется только один раз
- Напоминание не отправляется, если задача выполнена или уже просрочена

## Очерёдность действий при просрочке
- Сначала фиксируется просрочка задачи
- Затем применяется урон и обновляется статистика пользователя
- При смерти персонажа прогресс сбрасывается и активные задачи удаляются
- Пользователь получает отдельное уведомление о смерти

## Устойчивость к конкуренции
- Если пользователь выполняет задачу в момент проверки дедлайна, итог определяется по факту дедлайна
- Повторная обработка одной и той же задачи не приводит к повторному урону
